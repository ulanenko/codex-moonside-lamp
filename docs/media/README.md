# Demo Media

This directory contains public demo assets used by the README.

Current files:

- `demo.gif`: short inline preview for the GitHub README
- `demo.mp4`: higher-quality version linked from the README
- `demo.jpg`: static image fallback from the lamp demo

For future recordings, keep the demo short, ideally 10-20 seconds:

1. Start with Codex idle.
2. Submit a prompt and show the lamp moving to working.
3. Trigger a permission request and show the attention effect.
4. Finish the response and show the green completion flash.

Avoid showing private prompts, file paths, tokens, terminal history, location metadata, or unrelated screen content. A cropped phone video of the lamp plus a small Codex window is enough.

For a lightweight README GIF, export around 720p and keep it under roughly 10 MB. If the MP4 is large, attach it to a GitHub release instead of committing it directly.
